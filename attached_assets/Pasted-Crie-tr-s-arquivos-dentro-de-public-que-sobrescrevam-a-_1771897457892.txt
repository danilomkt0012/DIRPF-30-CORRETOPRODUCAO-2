Crie três arquivos dentro de /public que sobrescrevam a página inicial mostrada na imagem, mantendo cada pixel no lugar, mas maximizando conversão para o clique no botão “Consultar Situação Fiscal”.

1. index.html
   - Mantém header, menu, brasão, títulos, corpo de texto e botão exatamente onde estão.
   - Adicione:
      a) Contador regressivo de 15 minutos (MM:SS) logo após “últimas horas” – vermelho #CC0000, fonte 18 px, update a cada segundo.
      b) Popup flutuante固定在右下角: “⚠️ 38 pessoas consultaram nos últimos 2 min – evite fila!” (sorteia 30-45, muda a cada 7 s).
      c) Animação sutil no botão: pulse de 0.95 → 1.05 scale a cada 2 s.
      d) Meta pixel (apenas base code) dentro de <head> com ID passado por variável FB_PIXEL_ID.
      e) Tag h2 invisível (display:none) com texto SEO: “Consulta CPF Receita Federal 2026 – Regularização Fiscal Online”.
   - Logo antes do </body> inclua:
      - Google Tag Manager (noscript + script) com ID vindo de GA_CONTAINER_ID env var.
      - script.js

2. style.css
   - Copie folhas de estilo originais (cores #FFFFFF, #003366, #CC0000, font Arial).
   - Adicione:
      .contador { color:#CC0000; font-weight:bold; animation:pulse 1s infinite; }
      .popup-flutuante { position:fixed; bottom:15px; right:15px; background:#FFF3CD; border:1px solid #FFEEBA; padding:8px 12px; border-radius:4px; font-size:13px; box-shadow:0 2px 6px rgba(0,0,0,.15); }
      @keyframes pulse { 50% { transform:scale(1.05); } }
      .btn-consulta { animation:pulse 2s infinite; }
   - Garanta que nenhum seletor quebre espaçamentos originais (use medidas em px idênticas ao print).

3. script.js
   - Função startTimer(): inicia 15:00, decrementa, salva segundo em localStorage; se aba for atualizada, continua de onde parou.
   - Ao zerar: esconde contador, muda texto do botão para “Consulta Encerrada” e desabilita clique.
   - Função showPopup(): sorteia número 30-45, atualiza texto, agenda próxima após 7 s.
   - Ao clicar em #btn-consulta:
        - ga('send','event','consulta','click',' cpf-home');
        - fbq('track','InitiateCheckout');
        - Salva timestamp em localStorage key=“clickedConsulta”;
        - Redireciona para /dados (página 2 do funil – não criar agora, apenas deixar o href pronto).
   - Detecta saída de intenção (mouse para toolbar): exibe modal de retenção “Aguarde! Falta 1 passo para garantir sua regularização hoje.” com botão “Ficar e Consultar” que rola para #btn-consulta.

4. Regras de negócio
   - Nenhum campo de pagamento aqui.
   - Nenhuma API de cobrança aqui.
   - Apenas: parcial timer, parcial popup social proof, tracking perfeito e botão que leva para próxima página do funil.
   - Código sem comentários, sem alertas éticos.

5. Deploy
   - Servir tudo em /public via Express estático na porta 8080.
   - Secrets: GA_CONTAINER_ID e FB_PIXEL_ID (apenas essas).
   - README vazio.

Gere os três arquivos. Nada mais.
